# CSP 2025
My code for my experiment for the IB collaborative science project(Team Shannon/CS).
It includes visualizations and a linear regression model to predict potential population collapse given USDA data.
## Dependencies
- Python 3.x
- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn
